# Security Policy

## Project posture

`thinlet-modernized` is a modernization fork of the 2002–2005 Thinlet GUI
toolkit, published as a public artifact **without any support guarantee**.
Free; caveat emptor. There is no SLA and no commitment to triage issues from
third parties.

## Reporting

If you find a security issue, please open an issue on the GitHub repository:

> https://github.com/nomixer/thinlet-modernized/issues

Response is **best-effort only**. There is no embargo process, no bounty, and
no guaranteed response time. For anything sensitive, note that this is an
unsupported fork of legacy desktop-GUI code; evaluate suitability accordingly
before relying on it.

## Supported versions

Only the latest `main` is considered. Older tags are not patched.
